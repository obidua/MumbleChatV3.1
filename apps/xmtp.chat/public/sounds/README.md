# Notification Sounds

This directory contains notification sound files for the MumbleChat application.

## Files:

- `message-receive.mp3` - Sound played when receiving a message
- `message-sent.mp3` - Sound played when sending a message

## Note:

These are placeholder files. Replace them with custom sounds for your application.
The sounds should be:

- Short (< 1 second)
- Clear and pleasant
- Small file size (< 50KB each)

## Format:

- MP3 format is recommended for cross-browser compatibility
- Backup formats (OGG, WAV) can be added for broader support
